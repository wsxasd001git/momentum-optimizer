Скринер моментума для российского рынка акций с бэктестингом и рекомендациями.

== Описание ==

Momentum Screener - это WordPress плагин для анализа и скрининга российских акций на основе стратегии моментума.

**Основные возможности:**

* Расчет momentum для российских акций
* Настраиваемые параметры стратегии
* Полный бэктестинг с метриками
* Текущие рекомендации для покупки
* Визуализация результатов (графики)
* Поддержка дивидендов
* Фильтр волатильности
* Динамический режим адаптации к рынку

**Метрики:**

* Общая и годовая доходность
* Коэффициент Шарпа
* Коэффициент Сортино
* Максимальная просадка
* Волатильность портфеля

== Установка ==

1. Загрузите папку `momentum-screener-wp` в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Перейдите в Настройки > Momentum Screener
4. Укажите URL Google Sheets с данными о ценах акций
5. Добавьте шорткод `[momentum_screener]` на любую страницу

== Настройка данных ==

Для работы плагина необходим Google Sheets документ с ценами акций:

1. Создайте Google Sheets документ
2. Назовите первый лист "цены"
3. В первом столбце укажите даты (Time)
4. В остальных столбцах - тикеры с ценами
5. Опубликуйте документ в веб
6. Скопируйте URL для CSV экспорта

**Формат URL:**
`https://docs.google.com/spreadsheets/d/{ID}/export?format=csv&gid={SHEET_ID}`

== Использование ==

**Базовый шорткод:**
`[momentum_screener]`

**С параметрами:**
`[momentum_screener lookback="6" holding="1" topn="15" show_backtest="true"]`

**Параметры:**
* `lookback` - Период расчета momentum (1-12 месяцев)
* `holding` - Период удержания (1-6 месяцев)
* `topn` - Количество акций в портфеле (5-30)
* `show_backtest` - Показывать бэктест (true/false)

== Часто задаваемые вопросы ==

= Какой формат данных нужен? =

Excel или Google Sheets с:
- Первый столбец: Time (даты в формате YYYY-MM-DD)
- Остальные столбцы: тикеры акций с ценами
- Каждая строка = один месяц

= Как добавить дивиденды? =

Создайте отдельный лист "дивиденды" в том же формате и укажите URL в настройках плагина.

= Что такое фильтр Reversal Effect? =

Исключает последний месяц из расчета momentum, чтобы избежать краткосрочных разворотов.

= Как работает динамический режим? =

При высокой рыночной волатильности автоматически увеличивает диверсификацию портфеля.
